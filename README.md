# E-Plant Shopping

## Project Title

E-Plant Shopping Website

---

## Description

E-Plant Shopping is a modern and responsive web application that allows users to browse and purchase a variety of indoor and outdoor plants. The project provides an attractive user interface for exploring different plant categories, viewing product details, adding plants to a shopping cart, and creating a smooth online shopping experience.

### Features

- Browse different plant categories
- View plant details
- Add products to cart
- Responsive design
- User-friendly interface
- Fast and interactive shopping experience

---

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/Shreya-krishik/e-plantShopping.git
```

### 2. Navigate to the project directory

```bash
cd e-plantShopping
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The application will start locally. Open the URL displayed in your terminal (usually `http://localhost:5173`).
